Title: Unit testing in C++: The good, the bad and the ugly

Type: 40min talk

Tags: C++ testing

Abstract:

Status: maybe next time

C++ is a powerful language in the hands of a seasoned developer. It is being used everywhere from operating systems and compilers to mobile apps development.
But this power comes with a cost - programming is C++ is not simple and can be downright difficult at times – with memory leaks, stack overflows and strange exceptions a software developer needs all of the power he can use.
By harnessing the power of unit tests a software developer can find defects quickly, avoid regression and speed up development time.
In this talk I’ll show how to use the right unit testing tools and practices available and how to write robust, maintainable unit tests in C++.
